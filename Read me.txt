Read me

1. Завантажити файли з репозиторію за наступним посиланням https://github.com/Doctor777/mysite/archive/master.zip на комп’ютер, після чого завантажити їх на хостинг в папку /public_html та розархівувати. Це можна зробити, наприклад, за допомогою CPanel або ж по FTP. 

2.Далі потрібно надати завантаженим на сервер файлам права. Для файлів - 644, для папок - 755.

3.Після цього потрібно розгорнути дамп бази даних на хостингу. Для цього створюємо нову базу даних та додаємо до неї користувача з правами доступу. Коли все успішно, заходимо в phpmyadmin, вибираємо створену базу даних, та натискаємо “Імпорт”. У вікні, що з’явилося, вибираємо SQL файл для завантаження на хостинг, вибираємо кодування UTF-8, натискаємо ОК. Чекаємо повідомлення про успішне імпортування БД.

4.Налаштування бази даних повинні відповідати налаштуванням у файлі параметрів сайту (/config/db_params.php).

5.В файлі index.php кореня сайту вписати назву сайту, та відключити виведення помилок:
ini_set('display_errors', 0); 
//error_reporting(E_ALL); // вмикаємо відображення всіх помилок 

6. По замовчуванню логін адміністратора - admin пароль - root

7. При реєстрації користувача, аккаунт автоматично баниться, поки адміністратор в адмінпанелі не активує його і не назначить йому права.
